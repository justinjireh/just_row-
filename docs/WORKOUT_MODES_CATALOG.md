# Workout Modes Catalog

This document is the product catalog for the workout modes RowPlus should grow into.

## Core Modes

1. `Free Row`
   - Open-ended rowing with no target.
2. `Quick Start`
   - One-tap row using the last-used workout settings.
3. `Time Goal`
   - Row for a selected duration.
4. `Distance Goal`
   - Row for a selected distance.
5. `Calories Goal`
   - Row until a calorie target is reached.
6. `Stroke Count Goal`
   - Row until a stroke target is reached.

## Interval Modes

1. `Intervals`
   - Simple work/rest blocks.
2. `Time Intervals`
   - Timed repeats with recovery windows.
3. `Distance Intervals`
   - Distance repeats such as 250m, 500m, or 1k.
4. `Pyramid`
   - Expanding and contracting work blocks such as 1-2-3-2-1 minutes.
5. `Rate Ladder`
   - Guided stroke-rate changes by stage.
6. `Power Intervals`
   - Guided power-target work intervals.
7. `Split Pace Workout`
   - Hold a target split window over structured blocks.

## Endurance And Recovery Modes

1. `Endurance Row`
   - Longer steady-state rowing.
2. `Recovery Row`
   - Low-intensity recovery work.
3. `Warmup`
   - Short ramp-up before a harder workout.
4. `Cooldown`
   - Guided easy row at the end of training.

## Benchmark And Skill Modes

1. `2K Test`
   - Standard benchmark piece.
2. `5K Test`
   - Mid-distance benchmark row.
3. `Technique Drills`
   - Low-rate power, pause drills, ratio drills, and stroke-control drills.

## Custom And Library Modes

1. `Workout Library`
   - Browse all preset workouts by category.
2. `Custom Workout Builder`
   - Assemble workout blocks manually.
3. `Training Plans`
   - Multi-week plans that chain workouts together.

## Scenic And Route Modes

1. `Scenic Rivers`
   - Scenic background media plus route progress.
2. `Scenic Lakes`
   - A calmer scenic variant using the same system.
3. `Urban Waterways`
   - Canal and city-water themed routes.
4. `Route Challenge`
   - Named route experiences with completion targets.

## Recommended Build Order

Build these in this order for the best product momentum:

1. `Free Row`
2. `Time Goal`
3. `Distance Goal`
4. `Intervals`
5. `Workout Library`
6. `Scenic Rivers`
7. `Custom Workout Builder`
8. `Training Plans`

## Near-Term Priority Set

If only a small set is being built now, prioritize:

1. `Free Row`
2. `Time Goal`
3. `Distance Goal`
4. `Intervals`
5. `Scenic Rivers`
